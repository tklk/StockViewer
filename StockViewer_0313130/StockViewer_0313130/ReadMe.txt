================================================================================
    MFC 程式庫：StockViewer_0313130 專案概觀
===============================================================================

應用程式精靈已經為您建立了這個 StockViewer_0313130 應用程式。這個應用程式不僅示範了使用 MFC 的基本概念，也是您撰寫應用程式的起點。

這個檔案包含一份摘要，簡要說明構成 StockViewer_0313130 應用程式的
所有檔案，它們個別的內容。

StockViewer_0313130.vcxproj
    這是使用應用程式精靈所產生之 VC++ 專案的主要專案檔。它含有產生該檔案之 Visual C++ 的版本資訊，以及有關使用應用程式精靈所選取之平台、組態和專案功能的資訊。

StockViewer_0313130.vcxproj.filters
    這是使用應用程式精靈所產生之 VC++ 專案的篩選檔。檔案中含有您專案中檔案與篩選器之間關聯的相關資訊。這項關聯用於 IDE 中以顯示特定節點下具有類似副檔名之檔案的群組 (例如，".cpp" 檔案會與 "Source Files" 篩選器相關聯)。

StockViewer_0313130.h
    這是應用程式的主要標頭檔。
    它包含了特定標頭 (包括 Resource.h)，並宣告 CStockViewer_0313130App 應用程式類別。

StockViewer_0313130.cpp
    這是含有應用程式類別 CStockViewer_0313130App 的主要應用程式原始程式檔。

StockViewer_0313130.rc
    這是程式所用的所有 Microsoft Windows 資源的列表。它含有儲存在 RES 子目錄中的圖示、點陣圖和游標。您可以直接在 Microsoft Visual C++ 內編輯這個檔案。您的專案資源是在 1028。

res\StockViewer_0313130.ico
    這是用來做為應用程式圖示的圖示檔。這個圖示是包含在主要資源檔 StockViewer_0313130.rc 中。

res\StockViewer_0313130.rc2
    這個檔案包含 Microsoft Visual C++ 無法編輯的資源。您應該將所有資源編輯器無法編輯的資源放到這個檔案。

StockViewer_0313130.reg
    這是範例 .reg 檔案，說明架構將為您設定的登錄設定類型。您可以將這個當做 .reg
    檔案，與應用程式一起執行，或者乾脆將它刪除，然後依賴預設的 RegisterShellFileTypes 登錄。

對於主框架視窗：
    專案包括標準 MFC 介面。

MainFrm.h, MainFrm.cpp
    這些檔案包含了框架類別 CMainFrame，其衍生自
    CMDIFrameWnd 並控制所有 MDI 框架功能。

res\Toolbar.bmp
    這個點陣圖檔案是用來建立工具列的並排影像。
    初始工具列和狀態列會在 CMainFrame 類別中建構。請使用資源編輯器編輯這個工具列點陣圖，並更新 StockViewer_0313130.rc 中的 IDR_MAINFRAME TOOLBAR 陣列，以加入工具列按鈕。
/////////////////////////////////////////////////////////////////////////////

對於子框架視窗：

ChildFrm.h, ChildFrm.cpp
    這些檔案會定義並實作 CChildFrame 類別，該類別支援 MDI 應用程式中的子視窗。

應用程式精靈會建立一個文件類型及一個檢視表：

StockViewer_0313130Doc.h、StockViewer_0313130Doc.cpp - 文件
    這些檔案包含您的 CStockViewer_0313130Doc 類別。編輯這些檔案，可加入您的特殊文件類型並實作檔案儲存和載入 (透過 CStockViewer_0313130Doc::Serialize)。
    文件將具有下列字串：
        副檔名：      csv
        檔案類型 ID：        StockViewer0313130.Document
        主框架標題：  StockViewer_0313130
        文件類型名稱：       StockViewer_031
        篩選條件名稱：         StockViewer_0313130 Files (*.csv)
        檔案新簡短名稱： StockViewer_031
        檔案類型完整名稱： StockViewer_0313130.Document

StockViewer_0313130View.h、StockViewer_0313130View.cpp - 文件的檢視表
    這些檔案包含您的 CStockViewer_0313130View 類別。
    CStockViewer_0313130View 物件是用來檢視 CStockViewer_0313130Doc 物件。

res\StockViewer_0313130Doc.ico
    這是用來做為 CStockViewer_0313130Doc 類別之 MDI 子視窗圖示的圖示檔。這個圖示是包含在主要資源檔 StockViewer_0313130.rc 中。
